# React Widgets

## About

This is about React widgets implemented for `widgets-react-vs-solidjs`.

## Install

```shell
# Install
cd client/widget.react
npm install
```

## React Widget Tips

For widgets using React isn't a big deal...
See [Tips for SolidJS](../widget.solid/README.md) since it is much harder.
